# Docs

Project documentation for mavenn — a Brazil-first rental billing and utility coordination app.

## Folders

### `project/`
Core product and design documentation.
- **project-plan.md** — MVP scope, user journeys, personas, roadmap, metrics, and technical direction
- **design.md** — Visual design system, typography, spacing, component guidance, and UX patterns

### `sales/`
Materials for pitching and onboarding early users.
- **alex-pitch.md** — Pitch deck for initial landlord partner

### `wireframes/`
HTML wireframes covering all 9 MVP user journeys. Each file is standalone and can be opened directly in a browser.
- **screens/** — 46 individual screen wireframes (grayscale, mobile-first, 390px phone frame)
- **journeys/** — 9 journey storyboards that compose screens via iframes into left-to-right walkthroughs with narrative captions
