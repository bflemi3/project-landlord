# Docs

Project documentation for Mabenn — a Brazil-first rental billing and utility coordination app.

## Folders

### [`project/`](project/)
Core product and design documentation.
- [**Project Plan**](project/) — MVP scope, user journeys, personas, roadmap, metrics, and technical direction
- [**DESIGN.md**](../DESIGN.md) — App design system: tokens, typography, spacing, component patterns, UX (warm-dark editorial)

### [`sales/`](sales/)
Materials for pitching and onboarding early users.
- [**alex-pitch.md**](sales/alex-pitch.md) — Pitch deck for initial landlord partner

### [`wireframes/`](wireframes/)
HTML wireframes covering all 9 MVP user journeys. Each file is standalone and can be opened directly in a browser.
- [**screens/**](wireframes/screens/) — 46 individual screen wireframes (grayscale, mobile-first, 390px phone frame)
- [**journeys/**](wireframes/journeys/) — 9 journey storyboards that compose screens via iframes into left-to-right walkthroughs with narrative captions

### [`research/`](research/)
Customer-discovery materials for validating the audience and problem.
- [**Entrevista — Imobiliárias**](research/interview-imobiliarias.html) — bilingual (PT-BR / EN) discovery script; open on a phone or laptop during interviews

### [`design-references/`](design-references/)
Standalone HTML captures of the product UI — the visual north star for the app reskin. Open any screen directly in a browser.
- [**Revenue dashboard**](design-references/revenue-dashboard.html) · [**Charges ledger**](design-references/charges-ledger.html) · [**Contract timeline**](design-references/contract-timeline.html) · [**Reputation**](design-references/reputation.html) · [**Communication**](design-references/communication.html)
