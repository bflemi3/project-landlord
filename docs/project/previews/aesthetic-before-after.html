What changed, at a glance

Palette
zinc (cool) → stone (warm). Teal unchanged.
Type scale
Retire text-[17px] / text-[28px]. Use text-xl card titles, text-3xl page / amounts, never text-xs on mobile.
Card padding
p-5 → p-7. Section rhythm mt-8 → mt-10.
Radius
rounded-2xl (16px) → rounded-[20px]. Softer, more editorial.
Chrome
Per-row borders → grouped card with hairline dividers. Status dot + pill → inline tone-coloured text.
Money hierarchy
Amount is the visual anchor. text-3xl on the card, not text-xl. Context (Apr 2026) moves above the title as a quiet eyebrow.
Shadow / border
Light: soft shadow, no border — unchanged philosophy, just warmer cast. Dark: hairline (inset 1px, 4% white) over a half-step lighter surface.
Font
Inter, unchanged. Larger sizes, tighter tracking on display, two text colours per surface instead of four.
