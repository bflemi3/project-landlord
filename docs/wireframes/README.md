# Wireframes

Grayscale, mobile-first HTML wireframes covering all 9 MVP user journeys for mabenn. Each file is standalone and can be opened directly in a browser — no build step required.

Built with Tailwind CDN and Lucide icons, rendered in a 390px phone frame.

## Folders

### [`screens/`](screens/)
46 individual screen wireframes. Each screen is defined once and embedded via iframes into journey files.

### [`journeys/`](journeys/)
9 journey storyboards that compose screens into left-to-right walkthroughs with narrative captions and flow arrows.
