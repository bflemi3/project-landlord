Variable Charge & Bill Format

A landlord creates a variable charge, selects the matching bill format by previewing example documents, or requests a new format if none match. They're notified when the new format is ready.

1 Add Variable Charge

Create a new variable charge — name, category, provider, and responsibility. Amount comes from bills each month.

→
2 Select Bill Format

Browse available bill formats for the provider. Preview example documents to find the one that matches. Can also enter manually or request a new format if none match.

→
3 Request New Format

If no format matches: submit provider name, label, sample PDF, and optional notes. Team builds the extraction config.

→
4 Format Ready

Landlord receives notification when the bill format is ready. Can immediately upload the first bill for extraction.
