Journey 1

First-Time Landlord Setup

A new landlord signs in, creates a property, defines charges and responsibility, invites tenants, and is prompted for feedback on the setup experience.

1 Sign In

Landlord signs in with Google or email. First-time user flow begins.

→
2 Create Property

Enter address, name the first unit, select currency. Guided 3-step flow.

→
3 Set Up Rent

Define the fixed monthly rent amount, due day, and who pays.

→
4 Add Expected Charges

Add electric, water, gas, condo fee, internet. Mark each as fixed or variable.

→
5 Allocate Responsibility

For each charge, choose: tenant pays, landlord pays, or shared split with percentage.

→
6 Invite Tenants

Add tenant by email. They'll get access to view statements and mark payments. Can skip.

→
7 Feedback

"Was setting up this property easier than your usual process?" Trust pulse survey.
