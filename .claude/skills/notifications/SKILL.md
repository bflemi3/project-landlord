# Notifications

Support important workflow moments without creating noise.

## MVP Notifications

- Statement ready
- Statement updated
- Due reminders
- Extraction needs review
- Dispute opened
- Payment marked
- Payment confirmed
- Payment rejected
- New invoice profile ready

## Channels

- Email
- In-app notifications

Push can come later if clearly valuable.
